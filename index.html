Como configurar Gmail corporativo
  1. Ve a console.cloud.google.com
  2. Crea un proyecto nuevo → habilita Gmail API
  3. Credentials → Create → OAuth 2.0 Client ID → Web Application
  4. En "Authorized JavaScript origins" agrega tu URL de Vercel
  5. Copia el Client ID y pegalo arriba
  6. Haz clic en "Conectar Gmail corporativo"
